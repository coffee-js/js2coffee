Js2coffee 帮助您迁移项目到 CoffeeScript

介绍

Js2coffee 帮助您迁移项目到 CoffeeScript 通过编译你的 JavaScript 代码到 CoffeeScript.

Js2coffee Rico Sta Cruz 的一个项目. 代码在 GitHub 上可以看到.

本地使用

Js2coffee 同样是一个 Node.js 模块.

$ npm install js2coffee
$ js2coffee file.js > file.coffee

提交 issue

有遇到任何的 issue 请提交到Js2coffee issue 收集器. 阅读 贡献手册然后提交 issue.
